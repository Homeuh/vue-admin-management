## 电商后台管理系统

### 项目技术栈
Vue2 + Vue-router + vuex + Element-UI + axios + Echarts

### 后端API
[配套修改后的接口API](https://github.com/Homeuh/vueShop-api-server.git)

[原接口API](https://gitee.com/wBekvam/vueShop-api-server.git)
### 运行项目
- 运行后端仓库
  - git clone 后端仓库地址
  - cnpm install
  - node app
- 运行本仓库代码
  - git clone 本仓库地址
  - cnpm install
  - npm run serve
